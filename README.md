# SDR / Space Avionics Notes (Personal)

Personal knowledge base for SDR and spacecraft avionics concepts:
multi-SSDR setups, synchronization, QPSK, PRS, PPS, etc.

Designed to be opened as an **Obsidian vault**.

## How to use
- Clone repo locally
- Open folder as Obsidian vault
- Notes are Markdown + KaTeX + Mermaid

## Conventions
- **Data plane**: I/Q streams and DSP
- **Control plane**: configuration, orchestration, states (often backend software)
